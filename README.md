# Бот сигналов на лонг (крипта)

Telegram-бот, который сканирует рынок криптовалют сразу на **нескольких биржах**
(Binance, Bybit, Bitget, Gate.io, KuCoin), анализирует **любые** монеты из
нескольких источников данных и присылает сигналы на **лонг** с указанием
уверенности, бирж листинга, уровней входа, стопа и тейка.

> ⚠️ **Дисклеймер.** Это инструмент для анализа, а **не** финансовый совет и не
> гарантия прибыли. Крипторынок высокорискован. Торгуйте только теми средствами,
> которые готовы потерять. Все решения — на вашей ответственности.

---

## Как это работает

Бот построен по **модульной архитектуре**: каждый источник данных — отдельный
«коллектор», который выдаёт балл за/против лонга. Баллы взвешиваются и
складываются в общий скоринг → если уверенность выше порога, отправляется сигнал.

### Мульти-биржевое сканирование

Бот опрашивает публичные API нескольких бирж и собирает **объединённую
вселенную монет**: монета может торговаться сразу на нескольких площадках — их
объёмы суммируются, а свечи для анализа берутся с биржи с наибольшим объёмом.
Так покрываются монеты, которых нет на Binance.

Поддерживаются: **Binance, Bybit, Bitget, Gate.io, KuCoin** (задаются в `EXCHANGES`).

**Фильтр качества (`MIN_EXCHANGES`).** Монета проходит, если торгуется минимум на
N биржах ИЛИ листится на топ-бирже (Binance/Bybit). Это отсекает мусор вроде
токенизированных акций и монет с накрученным объёмом, которые встречаются лишь
на одной второстепенной бирже. Поставь `MIN_EXCHANGES=1` для максимального охвата.

### Источники данных

| Источник | Что анализирует | Статус |
|---|---|---|
| **Binance (Technical Analysis)** | Свечи, RSI, EMA9/21/50, MACD, объём | ✅ Бесплатно, из коробки |
| **Derivatives (Funding/OI)** | Funding rate, Open Interest, Long/Short — с публичного API фьючерсов Binance | ✅ Бесплатно, из коробки |
| **DefiLlama** | Рост/падение TVL сети монеты | ✅ Бесплатно, из коробки |
| **Fear & Greed** (alternative.me) | Настроение всего рынка | ✅ Бесплатно, из коробки |
| **CoinGlass** | Funding rate, деривативы | 🔑 Нужен `COINGLASS_API_KEY` |
| **CoinMarketCap** | Импульс цены 24ч/7д | 🔑 Нужен `COINMARKETCAP_API_KEY` |
| **Glassnode** | Баланс монет на биржах (on-chain) | 🔑 Нужен `GLASSNODE_API_KEY` |
| **CryptoQuant** | Netflow на биржи | 🔑 Нужен `CRYPTOQUANT_API_KEY` |
| **Nansen** | Потоки smart money | 🔑 Нужен `NANSEN_API_KEY` |
| **Arkham** | Перемещения крупных сущностей | 🔑 Нужен `ARKHAM_API_KEY` |
| **DeBank** | Активность в EVM-сетях | 🔑 Ключ (точка расширения) |
| **DropsTab** | Ближайшие разблокировки токенов | 🔑 Нужен `DROPSTAB_API_KEY` |

**Про TradingView и GMGN:** у них нет публичного API (парсинг против правил),
поэтому TradingView-аналитику (RSI/MACD/EMA) бот считает **сам** по свечам
Binance — это те же индикаторы. GMGN (Solana-мемкоины) даёт только неофициальный,
нестабильный API, поэтому в основную сборку не включён.

Источники по ключу **включаются автоматически**, как только ты добавишь ключ в
`.env`. Без ключа они просто не голосуют — бот продолжает работать на остальных.

## Установка

Нужен Python 3.10+.

```bash
pip install -r requirements.txt
```

## Настройка

1. Создай бота в Telegram у [@BotFather](https://t.me/BotFather) и получи токен.
2. Скопируй `.env.example` в `.env`:

```bash
copy .env.example .env      # Windows
# cp .env.example .env      # macOS/Linux
```

3. Впиши `TELEGRAM_TOKEN`. Опционально добавь API-ключи источников и подстрой
   параметры (порог, таймфрейм, веса) — см. комментарии в `.env`.

## Запуск

**Проверить логику без Telegram** (разовый скан в консоль):

```bash
python scan_once.py
```

**Запустить бота:**

```bash
python bot.py
```

Затем в Telegram открой своего бота и отправь `/start`.

### Команды бота

- `/start` — подписаться на сигналы
- `/stop` — отписаться
- `/scan` — просканировать рынок прямо сейчас
- `/status` — активные источники и настройки
- `/help` — справка

## Настройка стратегии

Основные параметры в `.env`:

- `EXCHANGES` — биржи через запятую (`binance,bybit,bitget,gate,kucoin`).
- `MIN_EXCHANGES` — минимум бирж для монеты (фильтр качества/мусора).
- `SCAN_MODE` — `top` (топ по объёму) или `all` (все пары к USDT).
- `MAX_SYMBOLS` — сколько монет проверять.
- `MIN_VOLUME_USDT` — минимальный суточный объём (фильтр неликвида).
- `INTERVAL` — таймфрейм (`15m`, `1h`, `4h`, `1d`…).
- `SIGNAL_THRESHOLD` — порог уверенности (0–100) для отправки.
- `COOLDOWN_MINUTES` — антиспам по одной монете.
- `WEIGHT_*` — веса источников в общем скоринге.

Логика технического сигнала (веса, RSI/EMA/MACD/ATR) — в `sources/binance_ta.py`.
Правило принятия решения — в `engine/scoring.py`.

## Структура проекта

```
bot.py               # Telegram-бот и планировщик
scan_once.py         # разовый скан в консоль (без Telegram)
config.py            # загрузка настроек из .env
indicators.py        # EMA / RSI / MACD / ATR
engine/
  scanner.py         # обход монет и сбор вкладов источников
  scoring.py         # итоговое решение и уверенность
sources/
  base.py            # базовый класс источника
  market.py          # агрегатор монет по всем биржам (вселенная)
  exchanges/         # клиенты бирж: binance, bybit, bitget, gateio, kucoin
  binance_ta.py      # технический анализ
  fear_greed.py      # индекс страха и жадности
  defillama.py       # TVL сетей
  coinglass.py       # деривативы (по ключу)
  coinmarketcap.py   # рыночные данные (по ключу)
  onchain.py         # Glassnode/CryptoQuant/Nansen/Arkham/DeBank/DropsTab
```

## Как добавить свой источник

1. Создай класс-наследник `DataSource` в папке `sources/`.
2. Реализуй `analyze_symbol(...)` (или `analyze_market(...)`), верни
   `Contribution(score=-1..1, weight, reason)`.
3. Зарегистрируй его в `sources/__init__.py` в `build_sources()`.
